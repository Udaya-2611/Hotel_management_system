Hotel Management System

Description

The Hotel Management System is a Python-based application designed to manage room bookings, check-ins, and check-outs efficiently. It stores customer and room details using CSV files, reducing manual work and improving accuracy in hotel operations.

Features

* View available rooms
* Book rooms easily
* Checkout system
* Stores customer details
* File-based data storage

Technologies Used

* Python
* CSV (File Handling)

How It Works

1. The system initializes room data
2. Users can view available rooms
3. Rooms can be booked by entering details
4. Checkout updates room availability
5. All bookings are stored for future reference

How to Run

1. Install Python
2. Save the code as `hotel.py`
3. Run the file:

   ```bash
   python hotel.py
   ```
4. Follow on-screen menu

Project Structure

* `hotel.py` → Main program
* `rooms.csv` → Room data
* `bookings.csv` → Booking records
